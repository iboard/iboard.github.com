iboard.github.com

github-pages made easy with rake .

You can fork this website and edit files in templates/ and pages/

Then run rake and push the repository to your own Github-page.
That easy.

The Project

This is the source to generate my github-page `iboard.github.com`.

See: Github Pages

Usage

  1. Fork from Github
  2. edit files in pages/*html.erb
  3. edit files in templates/* (optional)
  4. run rake will generate static web-pages and opens index.html
  5. run rake -T to see more rake-task
